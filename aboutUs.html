How it began

This project is the first time using unity for the four of us putting it together.
We have been working on it for about two weeks at this point in time, and the main
reason for starting it was school. We started it as our final project in our Object
Oriented Programming class. We were just throwing out ideas on what we could do,
whe Ashley threw out the idea of a dating simulator. At first it was kinda laughed
off until we started thinking about it a bit more. We realized that being in a school
filled with programmers, we could make it based on the personificatinos of some of
the popular programming languages. Coming up with how the characters could act, and
knowing that several people have already given many of the languages their own human
personality. Then one thing after another, here we are today!

Our Future Plans

Our plans for this project obviously include the things on our checklist that we have
not yet completed. However, it also includes several other things like potentially
selling the game to attemp to gain some money. This has not been decided for sure by
any means. If it does go through though, it would be nice to be able to get the game
released on the steam store allowing for easier access to the game.

Current Status

The game is currently entering a slower patch of development due to the fact that we
no longer need to be working on it for a grade. However it will very much still be in
development until it is in a complete state that we are all happy with. We are already
proud of ourselves for how far we have gotten, but we know we can, and will, do better.
